Vortex Police Report System .

Overview
VTX PD Report System is a comprehensive police documentation system for FiveM QBCore servers. It provides an immersive, professional interface for creating, managing, and storing police reports, academy evaluations, and official documents.

Key Features
Multiple Document Types: Internal Affairs, Police Academy Evaluations, Official Statements, Arrest Reports
Realistic Animations: Players hold clipboard and notepad props while writing
Character Name Integration: Auto-fetch nearby players with their RP names
Custom Folder System: Name and organize document folders
Persistent Storage: Documents saved as items in QB inventory
Beautiful UI: Leather-bound report book with floating notes animation
Arabic Support: Full RTL support and Arabic translations

Document Types
Internal Affairs Document - Investigation & complaint reports
Police Academy Evaluation - Training assessment with ratings
Official Statement - Witness & officer statements
Arrest Report - Arrest & incident documentation


نظام تقارير الشرطة 

نظام تقارير الشرطة VTX هو نظام متكامل ومصمم باحترافية لسيرفرات FiveM باستخدام فريمورك QBCore. يوفّر تجربة واقعية وغامرة للضباط في توثيق القضايا والتقارير الرسمية داخل السيرفر بطريقة منظمة وأنيقة

 نظرة عامة
النظام يتيح للجهات الأمنية إنشاء، إدارة، وأرشفة جميع أنواع التقارير والوثائق الرسمية بسهولة تامة، مع واجهة احترافية تعزز أجواء الرول بلاي وتضيف طابع رسمي وواقعي للشغل الأمني.

المميزات الرئيسية
انواع متعددة من الوثائق
يشمل الشؤون الداخلية، تقييمات أكاديمية الشرطة، البيانات الرسمية، وتقارير الاعتقال.

 رسوم متحركة واقعية
الضابط يمسك حافظة ومفكرة أثناء كتابة التقرير لزيادة الواقعية.

 تكامل أسماء الشخصيات
يجلب تلقائيًا أسماء اللاعبين القريبين بأسمائهم الحقيقية داخل الرول بلاي.

 نظام مجلدات مخصص
تقدر تسمي وتنظم مجلداتك عشان ترتّب الوثائق بطريقة احترافية.

 تخزين دائم
يتم حفظ الوثائق كعناصر داخل مخزون QB، وتقدر ترجع لها بأي وقت.

 واجهة جميلة ومميزة
تصميم دفتر جلدي فاخر مع تأثيرات ملاحظات عائمة تعطي طابع رسمي جذاب
